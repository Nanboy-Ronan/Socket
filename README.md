# Socket
 This repository provides basic API of Socket in Java and Python.

## TCP
### Java
```shell
cd Socket/Java/TCP
javac Client.java
javac Server.java
java Server
java Client
```
### Python
```shell
cd Socket/Python/UDP
```

## UDP
